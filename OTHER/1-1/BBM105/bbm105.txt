1. generation of computing (1950-1957) --> 
Commercial computers, symbolic programming languages, binary arithmetic
Vacuum tubes used to store data and programs
Each computer was multiple rooms in size
Computers were not very reliable
2. generation of computing (1957-1965)
Transistors, magnetic cores, first disks for mass storage
Dramatic reduction size
Icrease in reliability of computers, reduced cost of computers
High-level programming languages, First operating system
3. generation of computing (1965-1975)
Integrated curcuits rather than individual electronic components were used
Further reduction in size and cost of computers
First minicomputers, time-shares operating systems, software industry formed
First set of computing standarts for compability between systems
4. generation of computing (1975-1985)
Reduced to the size a typewriter
Large-Very lange scale integrated circuits, first microcomputer developed
Desktop and personal computers common
New types of software and of the software industry
Computer networks, emails, embedded systems, graphical user interfaces
5. generation of computing (1985-)
Ultra large scale integrated circuits, supercomputers, parallel processors
Integrated global telecomminications, wireless data comminications
Incorporating data,TV,fax, Internet, Laptops, World Wide Web
High resolution graphics, visualization, virtual reality
Massive storage devices, ubiquitous computing
